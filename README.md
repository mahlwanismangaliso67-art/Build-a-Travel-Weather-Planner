# 🌦️ Travel Weather Planner (KM + Bolt Edition)

A simple Python-based Travel Weather Planner that helps users decide whether it is safe or practical to travel based on distance (in kilometers), weather conditions, and available transport options.

This project simulates real-world commuting decisions using basic programming logic.

---

## 📌 Features

- Uses distance in kilometers (KM) distance_km = 15
 ride-share availability  

---

## 🧪 Example Logic

- 🌧️ Rain + no bike → walking not recommended  
- ☀️ Clear weather + bike → cycling possible  
- 🚗 Long distance + car or Bolt → travel possible  
- ❌ No transport available → travel not possible
